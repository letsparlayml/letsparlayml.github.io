How to use this site

What visitors should see first

Keep the front page focused on the most useful items: today’s game predictions, the movement of each prediction across the week, and a short list of your best props.

How weekly movement works

Each game card links to a detail page with a simple movement chart and a daily table. You only need to export one record per game per day.

How to update it

Replace the JSON files in the /data folder with your newest outputs, commit to GitHub, and the site updates automatically.
